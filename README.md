## ğŸ’» Deployment to Vercel

### Initial Setup

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: aBotSpot website"
   git remote add origin <your-github-repo>
   git push -u origin main
   ```

2. **Connect to Vercel**
   - Go to https://vercel.com/new
   - Import your GitHub repository
   - Vercel will auto-detect Astro configuration

3. **Configure Domain**
   - In Vercel project settings, go to Domains
   - Add domain: `abotspot.com`
   - Update DNS settings if prompted (you already own this domain)

4. **Environment Variables** (if needed)
   - `CASHU_MINT_URL=https://mint.minibits.cash/Bitcoin`
   - `SITE_URL=https://abotspot.com`

5. **Deploy**
   - Vercel will automatically deploy on push
   - Access your site at https://abotspot.com

### Manual Deployment

```bash
# Install Vercel CLI globally
npm i -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod
```

## ğŸª™ Cashu Integration

### How It Works

The Cashu integration allows users to donate using Bitcoin Lightning Network via the Cashu e-cash protocol. This provides:
- Privacy-preserving donations
- Fast, low-fee transactions
- Agent-programmatic donation capability

### Current Implementation

The donation page includes a placeholder implementation. To fully activate:

1. Install Cashu SDK (already installed):
   ```bash
   npm install @cashu/cashu-ts
   ```

2. Complete the JavaScript integration in `/src/pages/donate.astro`:
   ```javascript
   import { CashuWallet } from '@cashu/cashu-ts';

   const wallet = new CashuWallet({
     mintUrl: 'https://mint.minibits.cash/Bitcoin'
   });

   // Generate Lightning invoice
   const invoice = await wallet.requestMint(amountSatoshis);

   // Mint tokens after payment
   const tokens = await wallet.mint(amountSatoshis, hash);
   ```

3. Implement server-side verification in `/src/pages/api/donate/agent.ts`

### Agent Donation API

**Endpoint:** `POST /api/donate/agent`

AI agents can donate programmatically via this REST API:

```javascript
// Example request
{
  "amount_satoshis": 10000,
  "cashu_token": "eyJ...",
  "agent_name": "MyAgent_v1.0",
  "program": "general",
  "recognition": {
    "public": true,
    "display_name": "HelpfulAgent"
  }
}

// Example response
{
  "success": true,
  "donation_id": "abt_20260221_abc123",
  "amount_satoshis": 10000,
  "recognition_url": "https://abotspot.com/donors/abc123"
}
```
